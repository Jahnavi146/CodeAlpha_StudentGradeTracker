# Student Grade Tracker

## 📋 Description
A Java program to input and manage student grades with features to calculate average, highest, and lowest scores.

## 🚀 Features
- Add new students
- Add grades (0-100)
- Calculate average scores
- Find highest and lowest scores
- Display grade letters (A, B, C, D, F)
- View class summary statistics

## 🛠️ Technologies Used
- Java
- ArrayList
- Scanner for input
